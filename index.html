SecureCounsel NY

Legal AI workbench for New York law-firm workflows. Attorneys, paralegals, and firm admins can query case documents in natural language while RBAC, matter-level permissions, audit logging, and security checks are enforced server-side.

Agentic hybrid retrieval with Qdrant vector search, BM25 sparse search, RRF fusion, Cohere reranking, and evidence grading. Security pipeline for prompt-injection detection, PII redaction, query decomposition, faithfulness checks, and hallucination blocking. Built with Next.js, TypeScript, FastAPI, Qdrant Cloud, Cohere, Groq Llama 3.3 70B, Supabase, Vercel, and Render.
